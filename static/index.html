🐙 BaekjoonHub 연동

추천 · 코드 포함 · 빠름

BaekjoonHub로 GitHub에 올린 풀이를 한 번에 가져옵니다.

github.com → Settings → Developer settings → Personal access tokens → repo 권한

▶

BOJ 직접 가져오기

BOJ 아이디를 입력하면 맞은 제출 기록을 자동으로 가져옵니다.
세션 쿠키를 입력하면 코드까지 저장할 수 있습니다 (선택 사항).

F12 → Application → Cookies → www.acmicpc.net → OnlineJudge 값 복사
코드를 저장해야 나중에 AI 리뷰 요청이 가능합니다.

▶

Codeforces 가져오기

Codeforces handle로 AC 기록을 가져옵니다. 소스 코드까지 같이 가져오려면 본인 계정 API Key/Secret이 필요합니다.

입력칸을 비워두면 서버의 `.env`에 있는 `CODEFORCES_API_KEY`, `CODEFORCES_API_SECRET`을 우선 사용합니다.

▶ GitHub 자동 push (BaekjoonHub처럼 풀이를 GitHub에 올리려면 펼치세요)

Settings → Developer settings → Personal access tokens → Contents: Write 권한 필요
소스 코드가 있는 문제만 push됩니다 (API Key/Secret 필요).
